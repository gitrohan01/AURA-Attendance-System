Automated Unified Recognition-based Attendance

AURA is an IoT-driven system that combines RFID-based identity verification and facial recognition to automate student attendance in classrooms. It eliminates manual intervention, reduces human error, and improves the reliability of attendance records.

Teachers can start and end attendance sessions using RFID cards, and student presence is verified via facial data stored on an SD card within the ESP32-CAM module. The system uses Wi-Fi or LoRa to communicate securely with a Django web server that stores, analyzes, and visualizes attendance data for teachers and HODs.
