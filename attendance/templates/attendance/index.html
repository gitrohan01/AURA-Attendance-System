Welcome to AURA

Automated Unified Recognition-based Attendance (AURA) is a next-generation IoT-powered attendance management system designed to simplify classroom attendance using RFID + Face Recognition technology. Built with ESP32-S3, ESP32-CAM, PN532, and a robust Django backend.

About Project

AURA is built to automate the attendance marking process in educational institutions. Using NFC cards and facial verification, it ensures that each student’s attendance is authenticated securely and efficiently. The system allows teachers to manage sessions and students easily while HODs can monitor class-level and subject-wise attendance analytics through an integrated dashboard.

Hardware Configuration

  • ESP32-S3 DevKitC-1 — Main controller and communication hub
  • ESP32-CAM — Handles face recognition and image verification
  • PN532 — RFID/NFC reader for student and teacher identification
  • 2.4" SPI ILI9341 TFT Display — Touchscreen interface for teacher actions
  • NRF24L01 (optional) — Wireless communication for remote classroom connectivity
  • SD Card Module — Stores student image data locally
